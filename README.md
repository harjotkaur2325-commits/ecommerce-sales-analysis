# Ecommerce-sales-analysis
E-commerce sales data analysis using Python

## Objective
To analyze e-commerce order data and identify sales trends, customer behavior, and operational insights.

## Dataset
The dataset contains order-level information such as order date, region, product category, promotion details, and courier status.

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- MS Word

## Key Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Regional order analysis
- Promotion impact analysis
- Order fulfillment analysis

## Key Insights
- Orders are concentrated in a few high-performing regions.
- Promotions have a strong impact on sales.
- Sales show seasonal patterns.
- Most orders are successfully delivered.

## Business Recommendations
- Focus on high-demand regions.
- Improve promotion strategies.
- Enhance courier status tracking.
- Plan inventory based on seasonal demand.

## Files in This Repository
- `Ecommerce_Sales_EDA.ipynb` – Data analysis notebook  
- `Ecommerce_Sales_Analysis_Report.pdf` – Business report  
- `ecommerce_sales_dataset.csv` – Dataset
